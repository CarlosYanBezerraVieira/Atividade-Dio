h1 até h6

'h1 até h6' representam os cabeçalhos de diferentes níveis de importância em um documento ou seção. O h1 é usado para o título mais importante, enquanto o h6 é usado para títulos menos importantes, com cada nível de cabeçalho indicando uma hierarquia de subseções no conteúdo.

p

'p' é usado para representar um parágrafo de texto em um documento HTML.

mark

'mark' é usado para destacar ou marcar parte do texto em um documento.

small

'small' é usado para representar texto de menor importância ou tamanho reduzido em um documento.

i

'i' é usado para representar texto em itálico, que é tipicamente apresentado em uma fonte diferente da do texto normal.

u

'u' é usado para representar texto sublinhado, indicando que o texto deve ser sublinhado no navegador.

strong

'strong' é usado para representar texto com importância forte, indicando que o texto é semanticamente mais importante do que o texto circundante.

ol

  1. 'ol' é usado para criar uma lista ordenada de itens, onde cada item é numerado sequencialmente.

ul

  • 'ul' é usado para criar uma lista não ordenada de itens, onde cada item é precedido por um marcador que indica sua posição na lista.

li

  • 'li' é usado para representar um item em uma lista, seja em uma lista ordenada ('ol') ou em uma lista não ordenada ('ul').

    • a

    'a' é usado para criar um hiperlink para outras páginas da web, arquivos, locais dentro do mesmo documento ou endereços de email.

    hr

    'hr' é usado para representar uma mudança temática no conteúdo de um documento, separando diferentes seções.

    sub

    sub' é usado para representar texto subscrito, que é tipicamente apresentado um pouco abaixo da linha de base normal text.

    sup

    'sup' é usado para representar texto sobrescrito, que é tipicamente apresentado um pouco acima da linha de base normal.

    blockquote

    'blockquote' é usado para representar uma seção de texto citado de outra fonte.

    font

    'font' é usado para definir o tamanho, cor e tipo de fonte do texto em um documento HTML, mas seu uso é desencorajado em favor de estilos CSS.

    del

    'del' é usado para representar texto que foi removido de um documento, muitas vezes mostrado com uma linha através dele.

    abbr

    'abbr' é usado para representar uma abreviação ou acrônimo, geralmente exibindo uma dica de ferramenta com a versão completa quando o cursor é colocado sobre ele.
